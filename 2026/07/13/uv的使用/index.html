uv的使用

发表于2026-07-13|更新于2026-07-13
|浏览量:

uv 是目前 Python 社区备受瞩目的极速包管理工具。它不仅速度极快，还能极大地简化我们的工作流。本文将根据不同的使用场景，带你快速掌握 uv 的日常用法。

场景一：轻量级任务（替代传统 venv）

适用背景

当我们需要运行一些单文件脚本或处理小型任务时，通常只需要一个干净、用完即走的虚拟环境。

1. 安装与配置

如果你使用的是 Windows 系统，可以通过 PowerShell 快速安装，并将其添加到系统环境变量中：

2. 环境的生命周期管理

利用 uv，管理轻量级虚拟环境变得非常直观：

  • 创建环境：在当前目录下瞬间创建一个名为 .venv 的虚拟环境。

  • 激活环境：激活后，命令行提示符会带有当前环境名称的前缀。

  • 安装依赖包：享受极速的安装体验。

  • 退出与清理：
    退出当前环境只需执行 deactivate；如果任务结束不再需要该环境，直接暴力删除 .venv 文件夹即可，没有任何残留负担。
